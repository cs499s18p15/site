Installation

To start, you will need to have obtained the source code files via the GitHub repo. Additionally, you will need to have MySQL (via XAMPP, Homebrew or some other means), Node.js, and NPM (Node Package Manager) installed. These links are provided in the References tab.

Next, you will use the provided make_wallchartDB_SQL.sql file to set up the tables in the database

Now install the npm dependencies by running the command npm install

Start up MySQL database configured to serve on port 3307 (using XAMPP or for Mac w/ Homebrew MySQL brew services start mysql )

Create ENV file by running: cp .env.sample .env  then input your specific information for the sensitive variables

To serve at localhost 4200, run npm run dev
