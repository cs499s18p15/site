Design Considerations

The MySQL Migration

As you may have seen documented elsewhere on the site, about a third of the way through our project we were told that our database implementation was not supported by Toyota. We had implemented our database component as an unstructured, No-SQL database (MongoDB). Unfortunately Toyota does not use any No-SQL databases at the moment, so they would have trouble finding someone to support it. Our project manager, Troy, told us that in order for wallchart tool to be used internally, we had to migrate to a structured database system, such as MySQL. So that's what we did! The overall migration took about two weeks, but all things considered it was for the best.
