Design

Overview

Wallchart Tool as it stands is being built as a full-stack SPA with a MEVN stack (Mongo, Express, Vue.js, Node.js). The backend node server hosts an express API, and also serves the Vue app for browsers to connect to. The front end communicates to the backend via the express API routes, which allows the node server to execute corresponding blocks of code. Environment

In order to run Walchart Tool locally, you need to make sure you've installed the following:

  • Node.js (and npm) v8.9.4
  • MongoDB Community Edition v3.6.2
The rest of the dependencies will be downloaded via npm, by running the command 
 npm install
from the root wallchart directory.

If you do not wish to run Wallchart Tool locally, but rather just want to connect to an existing server running Wallchart Tool, you will only need an up-to-date internet browser. For the best experience, we recommend using Google Chrome to connect to Wallchart Tool.
